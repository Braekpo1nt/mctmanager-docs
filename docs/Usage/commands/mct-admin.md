# /mct admin

This command is used to perform actions related to [Admins](../admins/admins.md)

## Syntax

- `/mct admin <add|remove>`
    - `/mct admin add <player>`
        - Arguments:
            - `<player>` the player to add to the admins
        - adds the given player as an admin
        - If the player was a [participant](../teams/teams-and-participants.md#participants) they will be removed from their team
    - `/mct admin remove <player>`
        - Arguments:
            - `<player>` the player to remove from the admins
        - removes the given player from the admins
    - This is used to add or remove admins. See [Admins](../admins/admins.md) for a description of what they are.
