## Admins

Admins are administrators of MCT. The Admin role can be given using the [`/mct admin`](../commands/mct-admin.md) command. Admins are teleported alongside players to the games, and are given special displays to show the current state of the event/game. 

## Adding/Removing Admins

See [`/mct admin`](../commands/mct-admin.md) for details on adding/removing admins
